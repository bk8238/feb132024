Block Level element (tag)

Paragraph tag is a block level tag

Similar to p tag, div tag is also a block level element.

it is important to note that all block level element start on a new line.

h1-h6 are also an example of block level element

Inline element (tag)

These elements do not start on a new line. One of the example is span element.

Example here is this red span tag where we have used span element.

In above example we used strong tag to bold "red span tag". Also, we used em tag to italicize it as well.

Block level element are always parent and Inline Element are always children.

Use this Formula

You cannot place block level tags like p tag, div tag etc. inside inline element tag like span tag, strong tag, em tag etc.

So the basic Formula would always be:

"Block_Tag_start= (Inline_Tag_start) (/Inline_Tag_end)
/Block_Tag_end"
